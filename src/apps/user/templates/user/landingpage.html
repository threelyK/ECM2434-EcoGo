What is EcoGo?

A Web-application development project created by our small dev team called "Breadscrums".

What is our Goal?

The goal of our project, is to educate our users about sustainability concepts, as well as persuade them to take action about these pressing issues. Our project is built to do this through gamification.

What we are doing?

We are creating a card game through this web application, in which we are gamifying each card to educate the user about certain topics related to sustainability.

Current state of development?

This is currently the prototype version so many features are still in the process to be implemented, thank you for reading.
